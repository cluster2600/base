# Réunions

Ce dossier contient les comptes-rendus et relevés de décisions créés par l'assistant. Chaque document est un fichier markdown nommé par date et sujet.

## Convention de nommage

```
YYYY-MM-DD_titre-reunion.md
```

Pour un relevé de décisions ou un suivi consolidé:

```
YYYY-MM-DD_releve-actions.md
```

### Exemples

- `2026-03-15_comite-direction.md`
- `2026-03-18_point-projet-alpha.md`
- `2026-03-20_releve-actions.md`

## Organisation

Les documents sont classés par date. Pour retrouver un compte-rendu ou suivre les actions ouvertes, consultez le nom des fichiers. Vous pouvez aussi demander à l'assistant de lister l'historique ou de faire le point sur les actions.
