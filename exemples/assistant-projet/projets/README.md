# Projets

Ce dossier contient vos projets actifs. Il se remplit au fur et à mesure: l'assistant y crée un sous-dossier par projet.

## Structure

Chaque projet a son propre sous-dossier:

```
projets/
├── nom-du-projet/
│   ├── fiche-projet.md         Cadre, objectifs, jalons, parties prenantes
│   └── point-avancement_YYYY-MM-DD.md  Points d'avancement datés
```

## Convention de nommage

- Dossier projet: nom court en minuscules avec tirets (ex. `refonte-site/`)
- Points d'avancement: `point-avancement_YYYY-MM-DD.md` (date du point)

Quand un projet est terminé, son dossier est déplacé vers `archives/` après un bilan final.
