# Archives

Ce dossier contient les projets terminés. Quand un projet de `projets/` est clos, l'assistant propose un bilan final, puis déplace son dossier ici.

## Structure

Chaque projet archivé garde la structure qu'il avait dans `projets/`:

```
archives/
├── nom-du-projet/
│   ├── fiche-projet.md
│   ├── point-avancement_YYYY-MM-DD.md
│   └── bilan-final.md          Bilan de clôture
```

Les archives restent lisibles et consultables: l'assistant peut s'y référer pour un historique ou un retour d'expérience.
