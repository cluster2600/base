# Starter perso: votre point de départ

## Essayez en 30 secondes

1. Ouvrez **ce dossier** (pas la racine du dépôt) dans Claude Code ou Cursor.
2. Dites, mot pour mot: **«Quelles sont mes options?»**
3. Vous devriez voir: un accueil en une phrase, puis un petit menu (préciser le rôle de votre assistant, ajouter un process, importer vos documents existants) et la question «Que souhaitez-vous faire?». Rien n'est écrit ni envoyé sans vous.

Le plus petit BASE personnel possible: un assistant générique, prêt à recevoir VOTRE métier.
Copiez ce dossier où vous voulez (par exemple dans vos Documents), renommez-le, puis ouvrez-le
dans Cursor ou Claude Code.

## Démarrer

1. Copiez ce dossier hors du dépôt (clic droit → Copier, puis collez dans vos Documents).
2. Ouvrez-le dans votre outil IA (Cursor: *File → Open Folder*; Claude Code: `cd` puis `claude`).
3. Dites bonjour, puis précisez le métier de votre assistant: éditez
   `.ai/agents/mon-assistant/AGENT.md` (sa description et son `use_when`).
4. Pour convertir vos documents existants: «importer mes procédures existantes».

## Le palier honnête

Ouvert tel quel dans votre outil, c'est le **modèle** qui route en suivant les consignes
projetées (le fichier que Claude Code et Cursor lisent à l'ouverture): pratique, mais sans
garanties mécaniques. Pour le routage déterministe et les écritures validées, passez à la CLI
ou au serveur MCP: voir la documentation de démarrage de BASE («Faites installer BASE par
votre IA»).

> Les données et les contenus présentés ici sont illustratifs: remplacez-les par les vôtres.
