# Candidatures

Ce dossier contient les dossiers de candidature et les comptes-rendus d'entretiens. Il se remplit au fur et à mesure du processus de recrutement.

## Structure

Chaque candidat a son propre sous-dossier:

```
candidatures/
├── nom-du-candidat/
│   ├── profil.md               Résumé du profil (formation, expérience)
│   ├── entretien_YYYY-MM-DD.md Compte-rendu et évaluation d'entretien
│   └── notes.md                Notes complémentaires
```

## Convention de nommage

- Dossier candidat: prénom-nom en minuscules avec tirets (ex. `marie-dupont/`)
- Entretiens: `entretien_YYYY-MM-DD.md` (date de l'entretien)

## Confidentialité

Les informations contenues dans ce dossier sont confidentielles. Elles ne doivent être partagées qu'avec les personnes impliquées dans le processus de recrutement concerné.
