# Postes ouverts

Ce dossier contient les offres d'emploi en cours de recrutement. Chaque offre est créée par l'assistant lors du processus de publication.

## Convention de nommage

```
YYYY-MM-DD_titre-du-poste.md
```

Exemple: `2026-03-15_developpeur-backend.md`

Si une grille d'entretien est préparée pour ce poste, elle se trouve dans:

```
YYYY-MM-DD_titre-du-poste_grille.md
```

## Cycle de vie

1. L'offre est créée ici lors de la publication
2. Les candidatures associées sont stockées dans `candidatures/`
3. Quand le poste est pourvu, l'offre peut être déplacée ou archivée
