# Clients

Ce dossier contient les fiches de vos clients. L'assistant propose de créer ou mettre à jour une fiche quand vous créez un devis.

Chaque fiche client est un fichier markdown avec les coordonnées et l'historique des interactions.

## Structure d'une fiche

```
nom-du-client.md
├── Informations générales (nom, adresse, contact)
└── Historique (liste des devis créés pour ce client)
```
