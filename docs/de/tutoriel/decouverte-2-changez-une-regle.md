# Ändern Sie eine Regel und sehen Sie, wie sie befolgt wird

*⏱ ~10 Min · Modul 2/3, Lernpfad Entdeckung*

**Sie werden**: feststellen, dass der Assistent Ihren Dateien folgt und nicht einem versteckten Gedächtnis, bewiesen durch das ✅ weiter unten.
**Sie brauchen**: Modul 1 abgeschlossen, das Tourismusbüro von Veytaux in Ihrem Werkzeug geöffnet.
↻ **Erinnerung**: Wofür dient das routing, ohne nachzuschauen? (um die richtige Aufgabe je nach Absicht auszuwählen)

1. Öffnen Sie `infos/tarifs.md`. Ändern Sie den Preis der **Führung durch das alte Dorf** von 12 auf 14 CHF.
2. **Speichern** Sie die Datei (Cmd+S / Ctrl+S: Cursor speichert nicht immer von selbst).
3. **Öffnen Sie eine neue Konversation** (wichtig: siehe Warum).
4. Fragen Sie erneut: *«Wie viel kostet die Führung durch das alte Dorf?»*

✅ **Prüfen Sie**: Der Assistent nennt 14 CHF (den neuen Preis), nicht 12. Wenn er weiterhin 12 sagt, schauen Sie bei den Störungen nach.

💡 **Warum es funktioniert hat**: Der Assistent hat weder eine Datenbank noch ein verstecktes Gedächtnis, er LIEST Ihre Markdown-Dateien bei jeder Aufgabe. Ändern Sie die Datei, und sein Verhalten folgt. Die neue Konversation erzwingt ein erneutes Lesen; in der alten konnte er den alten Preis im Kopf behalten.

🔁 **Bei Ihnen**: Welche Zahl, Regel oder Information ändert sich in Ihrem Beruf regelmässig und sollte in EINER Datei leben, die man aktualisiert?

→ **Und jetzt**: [Modul 3: Ihr eigener Ordner](decouverte-3-votre-dossier.md): Sie verlassen das Tourismusbüro von Veytaux für Ihren eigenen Bereich.

🆘 **Häufige Störungen**: *Er sagt weiterhin 12 CHF*: (a) die Datei wurde nicht gespeichert; (b) Sie sind in derselben Konversation geblieben: öffnen Sie eine neue. *Sie finden tarifs.md nicht*: sie liegt im Unterordner `infos/`.
