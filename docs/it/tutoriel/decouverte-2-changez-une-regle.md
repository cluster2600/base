# Cambia una regola e guardala obbedire

*⏱ ~10 min · modulo 2/3, percorso Scoperta*

**Farai**: constatare che l'assistente segue i tuoi file, non una memoria nascosta, dimostrato dal ✅ qui sotto.
**Ti serve**: il modulo 1 completato, l'ufficio del turismo di Veytaux aperto nel tuo strumento.
↻ **Promemoria**: senza guardare, a cosa serve il routing? (a scegliere il compito giusto in base all'intenzione)

1. Apri `infos/tarifs.md`. Cambia il prezzo della **Visita guidata del vecchio villaggio** da 12 a 14 CHF.
2. **Salva** il file (Cmd+S / Ctrl+S: Cursor non salva sempre da solo).
3. **Apri una nuova conversazione** (importante: vedi Perché).
4. Richiedi: *«Quanto costa la visita guidata del vecchio villaggio?»*

✅ **Verifica**: l'assistente annuncia 14 CHF (il nuovo prezzo), non 12. Se dice ancora 12, vedi i guasti.

💡 **Perché ha funzionato**: l'assistente non ha né un database né una memoria nascosta, LEGGE i tuoi file Markdown a ogni compito. Modifica il file e il suo comportamento segue. La nuova conversazione forza una rilettura; in quella vecchia poteva tenere a mente il vecchio prezzo.

🔁 **Da te**: quale cifra, regola o informazione cambia regolarmente nel tuo lavoro e dovrebbe vivere in UN file che si tiene aggiornato?

→ **E adesso**: [Modulo 3: la tua cartella](decouverte-3-votre-dossier.md): lasci l'ufficio del turismo di Veytaux per il tuo spazio personale.

🆘 **Guasti comuni**: *Dice ancora 12 CHF*: (a) il file non è stato salvato; (b) sei rimasto nella stessa conversazione: aprine una nuova. *Non trovi tarifs.md*: è nella sottocartella `infos/`.
