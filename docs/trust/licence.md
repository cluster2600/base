# Ce que vous pouvez faire avec BASE

Vous avez le droit de copier, d'adapter et de redistribuer BASE, y compris pour un usage commercial: voici ce que cela autorise précisément, et à quelles conditions. Que vous veniez du secteur public ou que vous soyez indépendant, retenez d'abord que BASE repose sur une double licence, une pour le code, une pour les contenus, et que ces droits n'ont pas tout à fait le même périmètre selon le fichier.

## Code: Apache-2.0

Le code, les tests, les schémas et les packages sont sous Apache-2.0.

Cette licence convient à un framework technique: elle pose un cadre clair pour la réutilisation, la distribution, la modification et la contribution, avec une clause de brevet.

## Contenus: CC BY 4.0

La documentation, les agents, les skills, les exemples, les templates et les contenus pédagogiques sont sous CC BY 4.0.

Cette licence est faite pour diffuser une méthode: vous pouvez copier, adapter et redistribuer, y compris pour un usage commercial, à condition d'attribuer la source.

Attribution recommandée:

> BASE, Bâtir des Assistants avec une Structure d'Expertise, par AI Swiss, https://a-i.swiss

## Source légale

Le fichier `LICENSE` à la racine reste la référence légale du dépôt.

En cas d'ambiguïté:

- un fichier exécutable ou destiné à être importé par une machine relève d'Apache-2.0;
- un fichier destiné principalement à être lu comme documentation, méthode, exemple ou instruction d'agent relève de CC BY 4.0.
